> Basic Installation :

| Method    | Command
|:----------|:--------------------------------------------------------------------------------------------------|

> Manual Installation :
```bash
git clone https://github.com/ankushbhagats/tbackup
cd tbackup
apt install ./tbackup.deb
```

+ $ tbackup --configure
